# 前置摄像头圆环显示模块

一个专为三星Galaxy S24 Ultra OneUI 7.0设计的Magisk模块，可以在前置摄像头周围显示白色圆环效果。

## 功能特性

- ✅ 在前置摄像头周围显示白色圆环
- ✅ 专门适配三星Galaxy S24 Ultra OneUI 7.0
- ✅ 支持LSPosed框架
- ✅ 可自定义圆环大小、宽度和透明度
- ✅ 系统级覆盖层，一直显示
- ✅ 开机自动启动
- ✅ 完整的配置界面

## 系统要求

- **设备**: 三星Galaxy S24 Ultra (推荐)
- **系统**: Android 7.0 (API 24) 或更高版本
- **框架**: Magisk + LSPosed (推荐)
- **权限**: 悬浮窗权限

## 安装步骤

### 1. 准备工作
确保您的设备已经：
- 安装Magisk
- 安装LSPosed框架 (推荐)

### 2. 安装模块
1. 下载 `camera_ring_overlay_v1.0.zip`
2. 在Magisk中选择"模块" → "从本地安装"
3. 选择下载的ZIP文件
4. 等待安装完成
5. **重启设备**

### 3. 激活模块
1. 重启后打开LSPosed应用
2. 进入"模块"页面
3. 找到"摄像头圆环显示"并勾选
4. 选择作用域：`系统UI (com.android.systemui)`
5. 重启SystemUI或设备

### 4. 配置使用
1. 打开"摄像头圆环显示"应用
2. 授予悬浮窗权限
3. 开启"启用圆环显示"开关
4. 调整圆环大小、宽度和透明度
5. 使用"测试圆环显示"按钮进行测试

## 配置说明

### 圆环设置
- **圆环大小**: 50-200dp，默认120dp
- **圆环宽度**: 2-20dp，默认8dp  
- **透明度**: 50-255，默认200 (数值越大越不透明)

### 位置适配
模块会自动检测设备型号并适配前置摄像头位置：
- 三星S24 Ultra: 屏幕顶部居中，55dp偏移
- 三星S24系列: 屏幕顶部居中，50dp偏移
- 其他三星设备: 通用三星位置
- 非三星设备: 通用Android位置

## 故障排除

### 圆环不显示
1. 确认已授予悬浮窗权限
2. 检查LSPosed中模块是否激活
3. 确认作用域包含SystemUI
4. 尝试重启SystemUI或设备
5. 检查Magisk模块是否正常启用

### 位置不准确
1. 打开应用查看设备检测信息
2. 手动调整圆环大小和位置
3. 如果是非S24设备，位置可能需要微调

### 权限问题
1. 进入系统设置 → 应用管理
2. 找到"摄像头圆环显示"
3. 授予所有必要权限，特别是悬浮窗权限

## 技术原理

### 工作机制
1. **Magisk模块**: 提供系统级安装和开机启动
2. **LSPosed Hook**: Hook SystemUI进程启动覆盖层服务
3. **覆盖层服务**: 使用WindowManager显示系统级圆环
4. **位置适配**: 根据设备型号自动调整摄像头位置

### 文件结构
```
camera_ring_module/
├── META-INF/                 # Magisk安装脚本
├── module.prop               # 模块属性
├── service.sh               # 启动服务脚本
├── customize.sh             # 自定义安装脚本
├── system/app/              # 系统应用目录
└── app/src/                 # Android应用源码
    ├── main/java/           # Java源代码
    │   ├── XposedModule.java    # LSPosed模块主类
    │   ├── service/             # 服务类
    │   ├── utils/               # 工具类
    │   └── receiver/            # 广播接收器
    └── main/res/            # 资源文件
```

## 常见问题

**Q: 圆环显示位置不准确怎么办？**
A: 可以在应用中调整圆环大小，或联系开发者适配您的设备。

**Q: 模块占用多少系统资源？**
A: 模块非常轻量，几乎不占用CPU和内存资源。

**Q: 是否影响相机功能？**
A: 不会，模块只是显示覆盖层，不会干扰相机工作。

**Q: 如何卸载模块？**
A: 在Magisk中禁用或删除模块，然后重启设备即可。
